## Organización de ejercicios y capturas

Cada ejercicio debe estar en un archivo `.py` independiente y, si corresponde, debe incluir una captura de pantalla con el mismo nombre que el archivo, pero con extensión `.png`. Por ejemplo:

```
ejercicio1/
├── ejercicio1.py
├── ejercicio1.png
```

Esto facilita la identificación y revisión de cada ejercicio junto a su evidencia visual.

## Recomendaciones para trabajar con Python

- Asegúrate de tener **Python 3.12** o superior instalado (se recomienda usar [pyenv](https://github.com/pyenv/pyenv) para gestionar versiones).
- Instala las dependencias necesarias usando `pip install -r requirements.txt` si el archivo existe.
- Ejecuta los scripts con:
	```sh
	/Users/gonzaloalvarezcampos/.pyenv/versions/3.12.2/bin/python archivo.py
	```
- Si agregas nuevas dependencias, crea o actualiza un archivo `requirements.txt`.
- Usa entornos virtuales para aislar dependencias si es necesario.
